# Data Analysis Skill

## Overview

This skill enables comprehensive data analysis and visualization.

## When to Use

- Analyzing datasets
- Creating charts and visualizations
- Generating statistical reports
- Finding patterns and insights in data

## Usage

```
analyze_data(data, analysis_type="statistical", output_format="json")
```

## Parameters

- `data` (required): Input data in various formats
- `analysis_type` (optional): "statistical", "trend", "correlation", etc.
- `output_format` (optional): "json", "csv", "charts", etc.

## Returns

- Analysis results and insights
- Generated charts and visualizations
- Summary statistics
