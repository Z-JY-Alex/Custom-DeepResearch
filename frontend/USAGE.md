# 使用指南

## 快速启动

### 方法一：使用启动脚本（推荐）

**Windows用户：**
```bash
# 双击运行
start_frontend.bat

# 或在命令行中运行
.\start_frontend.bat
```

**Linux/Mac用户：**
```bash
# 给脚本添加执行权限（如果需要）
chmod +x start_frontend.sh

# 运行脚本
./start_frontend.sh
```

### 方法二：手动启动

1. **启动后端API**
```bash
cd /path/to/deepresearch
python backend/api/stream_api.py
```

2. **启动前端服务器**
```bash
cd frontend
python server.py
```

3. **访问界面**
在浏览器中打开：`http://localhost:3000`

## 界面说明

### 主要区域
- **聊天区域**：显示对话内容和AI回复
- **输入框**：输入问题和指令
- **状态指示器**：显示AI当前状态
- **侧边栏**：显示文件内容（点击文件卡片时打开）

### 消息类型
- **用户消息**：右侧紫色气泡
- **AI回复**：左侧白色气泡，支持流式显示
- **工具调用**：灰色框，显示AI使用的工具
- **文件操作**：黄色卡片，可点击查看文件
- **错误消息**：红色边框，显示错误信息

### 交互功能
- **流式输出**：AI回复逐字符显示
- **工具可视化**：清晰展示工具执行过程
- **文件预览**：侧边栏实时显示文件内容
- **用户问答**：弹窗形式的交互对话

## 常见问题

### Q: 连接失败怎么办？
A: 
1. 确保后端API正在运行（端口8000）
2. 检查防火墙设置
3. 查看浏览器控制台错误信息

### Q: 流式显示不正常？
A: 
1. 确认浏览器支持ReadableStream API
2. 检查网络连接稳定性
3. 尝试刷新页面重新连接

### Q: 文件操作不显示？
A: 
1. 确认后端返回正确的文件事件
2. 检查事件数据格式
3. 查看控制台是否有JavaScript错误

### Q: 用户交互弹窗不出现？
A: 
1. 检查浏览器是否阻止了弹窗
2. 确认交互事件格式正确
3. 查看控制台错误信息

## 自定义配置

### 修改API地址
编辑 `app.js` 文件：
```javascript
// 第32行左右
this.apiBaseUrl = 'http://your-api-server:8000/api/v1';
```

### 调整流式显示速度
编辑 `app.js` 文件：
```javascript
// 第420行左右
setTimeout(typeChar, 20); // 改为更小的值显示更快
```

### 修改主题颜色
编辑 `index.html` 的CSS部分：
```css
/* 修改渐变色 */
background: linear-gradient(135deg, #your-color1 0%, #your-color2 100%);
```

## 开发调试

### 启用调试模式
在浏览器控制台中输入：
```javascript
// 查看所有事件
app.debugMode = true;

// 查看连接状态
console.log('连接状态:', app.isConnected);

// 查看当前会话
console.log('会话ID:', app.currentSessionId);
```

### 查看网络请求
1. 打开浏览器开发者工具（F12）
2. 切换到"网络"标签
3. 发送消息查看API请求和响应

### 查看控制台日志
所有重要事件都会在控制台输出日志，便于调试和排查问题。
